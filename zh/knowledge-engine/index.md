# 知识引擎

可插拔、可嵌入的知识引擎，用于文档摄入、分块、嵌入和混合搜索。

## 概述

知识引擎是一个独立服务，设计用于集成到任何项目中。它提供全面的 RAG（检索增强生成）功能，支持多种格式、嵌入提供商和存储后端。

## 核心特性

- **混合搜索** — 向量相似度（pgvector）+ BM25 全文搜索，通过 RRF 融合
- **中文支持** — 运行时 zhparser 检测，支持中文全文搜索
- **多格式解析** — PDF、DOCX、PPTX、XLSX、CSV、HTML、JSON 及源代码
- **可插拔嵌入** — OpenAI、Gemini、Ollama（兼容 OpenAI）
- **可插拔存储** — 本地文件系统、MinIO/S3
- **可插拔认证** — JWT、无认证（开发环境）
- **行业模板** — 可配置的元数据模式
- **RBAC 权限** — 用户、团队和组织级别的访问控制

## 使用场景

- **RAG 应用** — 为 LLM 提供来自自有文档的上下文
- **文档搜索** — 跨文档集合的全文和语义搜索
- **知识管理** — 从多个来源组织和检索信息
- **聊天机器人增强** — 基于事实文档的聊天机器人响应
