## 项目关系

### OmniDev 平台 + 知识引擎

平台集成了知识引擎作为其 RAG 后端：

- **文档管理** — 通过平台 UI 上传和管理文档
- **搜索集成** — 搜索结果输入到 AI Agent 上下文
- **共享基础设施** — 使用相同的 PostgreSQL 和 Redis 实例

### 独立使用

每个项目都可以独立使用：

- **知识引擎** — 作为任何应用的独立 RAG 服务
- **平台** — 通过 API 使用自定义 RAG 后端

## 技术栈对比

| 组件 | 平台 | 知识引擎 |
|------|------|----------|
| 语言 | Go | Go |
| HTTP 框架 | Gin | chi |
| 数据库 | PostgreSQL + pgvector | PostgreSQL + pgvector |
| 缓存 | Redis | Redis |
| 存储 | MinIO | 本地 / MinIO / S3 |
| 嵌入 | OpenAI / 自定义 | OpenAI / Gemini / Ollama |
| 前端 | Next.js | 无（仅 API） |
| 工作流 | Temporal | 无 |

## 使用场景

### 平台

- **AI 驱动开发** — 带 AI 辅助的 IDE
- **团队协作** — 共享工作空间和项目
- **工作流自动化** — 可视化工作流构建器
- **一键部署** — 部署到多个环境

### 知识引擎

- **RAG 应用** — 为 LLM 提供上下文
- **文档搜索** — 全文和语义搜索
- **聊天机器人增强** — 基于文档的响应
- **知识管理** — 组织和检索信息
